Sobre o PHPtest

Este projeto é realizado como teste técnico para o processo seletivo da empresa CD2 Serviços em Tecnologia da Informação Ltda.

Sendo este um serviço para recuperar endereço apartir de CEP.

Como usar:

Para usar o PHPtest é muito simples, basta digitar um CEP válido logo abaixo e pressionar ENTER, TAB ou o Botão . Em seguinda os campos do fomulário serão preenchidos automaticamente.

Se quiser acompanhar os que está acontecendo, fique de olho nas menssagens que aparecem acima.
